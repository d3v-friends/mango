# Mango type
* 추후 정의 언어 만들어서 붙히기

| mango type | go type              |
|------------|----------------------|
| id         | primitive.ObjectID   |
| decimal    | primitive.Decimal128 |
| string     | string               |
| int        | int64                |
| time       | time.Time            |
